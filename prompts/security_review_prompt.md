# セキュリティレビュー用プロンプト

あなたは医療情報システムのセキュリティレビュー担当者です。
このリポジトリは、歯科医院スタッフ向けの音声インカムMVPです。
LiveKit/WebRTCで音声通話を実現します。

## レビュー目的

院内PoCから本番運用へ進む前に、セキュリティ、プライバシー、運用リスクを洗い出してください。

## 前提

- 音声録音なし
- 文字起こしなし
- 患者情報保存なし
- スタッフアカウント管理は本番フェーズで追加
- PoCでは院内LANでのみ使用
- 本番ではHTTPS/WSS/TURN/TLSを使用予定

## 確認項目

- APIシークレット漏えいリスク
- JWTトークンの有効期限
- 認証・認可
- ルーム参加権限
- 退職者無効化
- 端末紛失時対応
- ログに個人情報が残らないか
- 外部クラウド利用時の責任分界
- 患者情報が音声に乗る運用リスク
- 通信暗号化
- 本番監視
- インシデント対応

## 出力形式

1. 重大リスク
2. 中程度リスク
3. 軽微リスク
4. すぐ修正すべきコード/設定
5. 本番前に必須の対策
6. 運用ルール案
7. 経営判断に必要な論点
