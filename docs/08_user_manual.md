# 08 スタッフ向け操作マニュアル（MIRISE Intercom）

このマニュアルは、院内スタッフが **MIRISE Intercom（院内音声インカム）** を使うための手順書です。
むずかしい知識は不要です。スマホとイヤホンがあれば使えます。

---

## 1. 用意するもの

- スマホ（iPhone / Android）または PC
- イヤホン（**通話対応（マイク付き）**のもの。ハウリング防止のため必ず着用）
- 医院の共通パスワード（管理者から教えてもらってください）

---

## 2. アプリを開く

1. スマホのブラウザ（Safari / Chrome）で次のURLを開きます。

   ```
   https://mirisevoicelink.vercel.app
   ```

2. よく使う場合は**ホーム画面に追加**しておくと便利です。
   - iPhone: 共有ボタン（□↑）→「ホーム画面に追加」
   - Android: メニュー（⋮）→「ホーム画面に追加」

---

## 3. ログインする

1. ログイン画面が表示されたら、**医院の共通パスワード**を入力して「ログイン」
2. パスワードが分からない場合は管理者に確認してください
3. ログインは約12時間有効です。切れたら再度ログインしてください

---

## 4. 通話に参加する

1. **スタッフ名**を入力（例: `佐藤` / `DH田中`）
   - 管理者がスタッフ登録をしている場合は、入力中に候補が表示されます
2. **参加ルーム**を選ぶ(受付 / 診療室 / オペ / 滅菌 / 全体 など)
3. **「接続する」**をタップ
4. 「マイクの使用を許可しますか?」→ **必ず「許可」**
5. 右上が**「接続中」(緑)**になれば参加完了です

> 💡 参加中のメンバーは画面下の **Participants** に表示されます。

---

## 5. 話し方（2通り）

### 方法A: 押して話す（基本・おすすめ）
- 画面の大きな**「押して話す」ボタンを押している間だけ**、あなたの声が流れます
- 指を離すとマイクOFF。トランシーバーと同じ感覚です
- PCでは**スペースキー長押し**でも話せます

### 方法B: タップで送信（ハンズフリー向け）
- **「● タップで送信開始 / 停止」**を1回タップ → 送信開始（マイクON）
- もう1回タップ → 送信停止（マイクOFF）
- **送信中は画面上部に赤い「🔴 送信中」バナー**が出ます。
  **話し終わったら必ず停止**してください（付けっぱなし厳禁）

> 🔘 キーを送るタイプの**BLEリモコン**（指輪型・クリップ型）を接続している場合、
> Enter・矢印・ページ送りキーで送信ON/OFFできます。
> 画面下の「リモコン/ボタンの確認」で、お手持ちのリモコンが使えるか確認できます。

---

## 6. ルームの切り替え・緊急呼び出し

- 画面のルームボタン（受付・診療室・オペ…）をタップすると、そのルームに移動します
- **「緊急 全体呼び出し」**（赤いボタン）を押すと、全体ルームに切り替わり、
  全員に緊急シグナルが送られ、数秒間マイクがONになります。**緊急時のみ**使ってください

---

## 7. 終了するとき

1. **「切断する」**をタップ（通話から抜けます）
2. 共有端末の場合は右上の**「ログアウト」**も押してください

---

## 8. 会話のルール（重要・医療機関として厳守）

- 🚫 **患者様の氏名・個人情報を声に乗せない**
  - ×「田中様が受付に」 → ○「**チェア3番**の準備お願いします」「**アポ枠2番**ご案内です」
- 🚫 録音・録画はしない（アプリにも録音機能はありません）
- 🎧 イヤホンを必ず使う（スピーカーだと患者様に聞こえます／ハウリングします）
- 🔴 「送信中」の赤バナーが出ている間は全員に聞こえています。消し忘れに注意

---

## 9. よくあるトラブル

| 症状 | 対処 |
|---|---|
| 声が聞こえない | イヤホンの接続を確認 → 一度「切断する」→ 再接続 |
| 自分の声が届かない | マイク許可を確認（ブラウザ設定）。イヤホンがマイク非対応の場合はスマホ本体のマイクで話す |
| 「接続に失敗しました」 | Wi-Fi/電波を確認して再接続。続く場合は管理者へ |
| ログインできない | パスワードを管理者に確認（大文字小文字も正確に） |
| 画面を消すと切れる | 仕様です（ブラウザ版は画面ONで前面のときだけ動きます）。ネイティブアプリ版で改善予定 |
| キーンと音が鳴る | ハウリングです。イヤホンを着けるか、近くの端末と距離を取ってください |

困ったら管理者（またはICT担当）へ連絡してください。
